# SceneAgent

AI-concierge 3D real-estate listings. A realtor records a walkthrough, drops freeform notes at timestamps, and SceneAgent builds an interactive Gaussian-splat listing with hotspots and a chat agent that can answer, measure, and guide a tour through the 3D scene.

See `docs/superpowers/specs/2026-04-20-sceneagent-design.md` for the full design spec and `docs/superpowers/plans/2026-04-20-sceneagent-plan.md` for the current execution plan. Track D will expand this README with architecture, metrics, and quick-start instructions.
