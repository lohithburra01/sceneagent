# sceneagent (api)

FastAPI backend for SceneAgent. Provides scene/object/note/hotspot REST endpoints,
an MCP server, and a LangGraph agent with Gemini Flash + VLM grounding.

See `docs/superpowers/specs/2026-04-20-sceneagent-design.md` (§4, §6, §9) and
`docs/superpowers/plans/2026-04-20-sceneagent-plan.md` (§Track B) in the repo root.
